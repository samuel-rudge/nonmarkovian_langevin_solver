# BUILD

This project is a pure Python codebase and does **not** require compilation.  
Building the project consists of setting up a suitable Python environment with the required dependencies.

---

## Requirements

- Python >= 3.9 (recommended)
- A Unix-like environment (Linux / macOS recommended)
- Sufficient disk space for trajectory and steady-state data

The code is developed and tested assuming a standard scientific Python stack.

---

## Environment Setup

It is **strongly recommended** to use a virtual environment or Conda environment.
